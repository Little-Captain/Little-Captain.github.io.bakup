    Module

    导入模块

    • 一个文件就是一个模块, 模块名就是文件名
    # as 使用自定义的名字引用这个模块
import modulename [as name]
# 从模块中引入指定的函数、类、变量等
from module import function1, function2...
# 从模块中引入所有的内容, 这时直接使用, 而不需要使用 . 引用
from module import *
    • 注意 : 通常, 导入模块的需要内容和功能, 而要避免使用 import *

    Python 自带模块

    • 查看 Python 环境 Path
