Module

导入模块

  • 一个文件就是一个模块, 模块名就是文件名
# as 使用自定义的名字引用这个模块

from module import function1, function2...

from module import *
  • 注意 : 通常, 导入模块的需要内容和功能, 而要避免使用 import *
